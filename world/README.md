# world/ — output generato

I file in questa cartella sono generati da `tools/build_world.py`.
**Non editare a mano.** Per modificare il mondo: aggiorna `content/` e rirun `make world`.

- `avatars/` — schede dei tre fratelli (vessel lettori-bambini).
- `characters/` — schede abitanti, divise per categoria (primari, testimoni, secondari, cuccioli).
- `_schema/` — JSON Schema del frontmatter.
- `_index.json` — indice flat per la web app.
